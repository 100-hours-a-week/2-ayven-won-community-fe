## 🚀 시작 방법

### 1. Git 클론
```bash
git clone https://github.com/100-hours-a-week/2-ayven-won-community-fe.git
cd community_ayven/front
```

### 2. 의존성 설치
```bash
npm install
```

### 3. 실행
```bash
npm start
```

### 4. 브라우저에서 열기
```
http://localhost:3000
```
